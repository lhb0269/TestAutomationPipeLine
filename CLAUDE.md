##프로젝트 목적
https://fconline.nexon.com/main/index 해당 페이지에서 AI를 활용하여
**Core Features:**
- 일반 로그인 (ID + Password)
- 입력값 유효성 검증
- 에러 메시지 처리
- 성공 시 페이지 이동 (https://fconline.nexon.com/main/index#today)
**Optional Features:**
- SNS 로그인 (카카오, 네이버)
위 Features 테스트 케이스 도출 ➝ 자동화 구현이 목표

**필수 산출물**
1. AI 활용 - 프롬프트 로그 사용 도구, 프롬프트, AI 응답 전체
2. 설계 - 테스트 케이스 정상/비정상/예외 시나리오
3. 개발 - 자동화 스크립트 Java + Selenium, 3개 이상 시나리오
4. 검증 - 실행 결과 콘솔 출력, 스크린샷
5. 분석 - 개선점 도출 AI 한계점, 보완 내용, 추가 케이스

##기술 개요
PlayWright MCP Server 활용
Test Case 기반으로 자동화 스크립트 제작 후 실행 영상 확보
실행 부터 결과 도출까지의 보고서 및 분석서 작성
